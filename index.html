What is UTF-8?

UTF-8 or UCS Transformation Format 8 is a standardized character encoding widely used in the web. Character encoding is the method computers used to store characters as data. All text in the web page is a sequence of characters stored in 1 or more bytes. A byte consists of 8 bits.
